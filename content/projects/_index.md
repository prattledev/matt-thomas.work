## RavennaCalc

A clean, browser-based bandwidth calculator for Ravenna/AES67 audio streams. Built to replace a clunky spreadsheet used when planning temporary WAN services for major sports events - where fibre is expensive and over-provisioning costs money.

## Comrex Fleet Dashboard

A lightweight Node.js dashboard for monitoring a fleet of Comrex broadcast audio codecs. Polls the Comrex Switchboard API and presents registration status, connection state, firmware version, and IP address for every unit in a single auto-refreshing view. Useful for pre-show checks and live operational monitoring across multi-site deployments.

![Comrex Fleet Dashboard](/assets/comrex_dashboard.png "Comrex Fleet Dashboard")

[Source on GitHub](https://github.com/prattledev/comrex-dashboard)

## GLITS & BLITS Generator

A web-based generator for broadcast-standard test tone sequences. Produces GLITS (the BBC/EBU stereo alignment cycle) and BLITS (the EBU 5.1 surround channel identification sequence) as 24-bit PCM WAV files at 48 kHz, with correct WAVE_FORMAT_EXTENSIBLE headers. Built for audio engineers who need compliant alignment tones without relying on a DAW or dedicated hardware. Python backend, vanilla HTML/CSS/JavaScript frontend, deployable via Docker.
